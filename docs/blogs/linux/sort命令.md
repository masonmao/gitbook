# sort命令

| 参数 | 说明                                      |
| ---- | ----------------------------------------- |
| -c   | 检查文件是否已经按照顺序排序              |
| -b   | 忽略每行前面开始出的空格字符              |
| -f   | 排序时，忽略大小写字母                    |
| -M   | 将前面3个字母依照月份的缩写进行排序       |
| -n   | 依照数值的大小排序                        |
| -o   | <输出文件> 将排序后的结果存入指定的文件   |
| -r   | 以相反的顺序来排序                        |
| -t   | <分隔字符>   指定排序时所用的栏位分隔字符 |
| -k   | 选择以哪个区间进行排序                    |
